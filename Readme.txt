- https://dev.twitch.tv/ 트위치 개발자 페이지 접속 후 

- 오른쪽 위에있는 ‘Login to twitch’ 버튼을 클릭하여 로그인

- 오른쪽 위에 있는 ‘Your Console’버튼을 클릭하여 콘솔 접속

- 왼쪽 사이드 메뉴에 있는 ‘응용 프로그램’메뉴 클릭

- ‘내 응용 프로그램 등록’ 클릭

- 이름에는 자신이 원하는 이름 (ex. 위버의 커스텀채팅창) 입력,

- OAuth 리디렉션 URL에는 ‘http://localhost’ 입력

- 범주는 Chat Box 선택 후 만들기 클릭

- 만들어진 애플리케이션의 우측 관리버튼 클릭

- 신규 시크릿 버튼 클릭

- 후에 프로젝트 폴더 안에있는 config.js 메모장으로 실행 후

broadcasterId: “[본인의 트위치 아이디]”,
clientId: “[clientId]”,
clientSecret: “[clientSecret]”
(대괄호를 빼고 입력해주세요)

입력 후 저장

OBS실행 후 브라우저 추가한 다음 로컬 파일 체크 후 프로젝트 안쪽 main.html 선택 후 확인

정상적으로 실행되는지 확인!!
